# Route Guardian

## Project Overview

Route Guardian is a backend service 
designed to inform users if the entirety 
or specific section of their 
route will encounter adverse weather.

By checking precise waypoints along a journey, 
it provides localised weather warnings. 
This is particularly useful for motorcyclists, 
cyclists, or anyone planning long-distance 
outdoor routing.

## Tech Stack

 - **Framework:** Spring Boot
 - **Database:** PostgreSQL (Using Docker Container)
 - **ORM:** Spring Data JPA / Hibernate
 - **Third Party:** Open-Meteo REST API (for weather data)
